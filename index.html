Meet Sunplate Free and Open-Source Toolkit

High Productivity

Sunplate is a set of utilities that let you develop apps in Go blazingly fast. Focus on things that matter, we'll take care of the rest.

High Performance

Code generation and good default components (such as denco router based on double-array) let you build apps that are very fast.

Type Safety

Catch errors when you are compiling your app not during runtime. The way Sunplate works will help you identify problems in code early.

100% Customizability

It is possible to customize every single thing. Wanna bring your own router, template system, or any other component? That's easy.

Getting started

{% highlight bash %} # Step 1: Install sunplate toolkit. go get -u github.com/anonx/sunplate # Step 2: Create a new skeleton application. sunplate new github.com/$username/$project # Step 3: Start a watcher / task runner. sunplate run github.com/$username/$project # Step 4: Start making changes to the app # generated at Step 1. {% endhighlight %}
