Meet Goal
Free and Open-Source Toolkit

High Productivity

Goal is a set of utilities that let you develop apps in Go blazingly fast. Focus on things that matter, we'll take care of the rest.

High Performance

Code generation and good default components (such as denco router based on double-array) make your web applications very fast.

Type Safety

Catch errors when you are compiling your app not during runtime. The way Goal works will help you identify problems in code early.

100% Customizability

It is possible to customize every single thing. Wanna bring your own router, template system, or any other component? That's easy.

Compatibility

Instead of reinventing the wheel Goal relies on builtin go generate subcommand and is fully compatable with the standard library.
