```sh
# Step 1: Install sunplate toolkit.
go get -u github.com/anonx/sunplate

# Step 2: Create a new skeleton application.
sunplate new github.com/$username/$project

# Step 3: Start a watcher / task runner.
sunplate run github.com/$username/$project

# Step 4: Begin making changes to the app
# generated at Step 1.

# Step 5: Use `sunplate help` to get more
# info about supported commands and
# `sunplate help {command}` to find out
# more about a specific command.
```
